# 巴克球-多语言对比学习系统
> 巴克球-多语言对比学习系统的目的是，期望在学习开发语言的时候，通过多种语言同步对比学习，达到相互印证，触类旁通，同时掌握多种开发语言的目的。比如进行Android开发，需要掌握Java、Kotlin、dart、Groovy、C++ 等，通过同步对比学习，就可以花最少的时间，掌握更多的语言。

Buckyball Mult Language Study System   (BMLSS)
